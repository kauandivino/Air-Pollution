# Air Pollution Extreme Event Benchmark

This repository contains a reproducible machine-learning pipeline for early detection of
future extreme air-quality events using a global monthly air-pollution benchmark. The
project formulates the task at the city-month level and evaluates how well models trained
on some countries transfer to unseen countries.

The main target used throughout the pipeline is whether a future AQI value crosses an
extreme threshold:

```text
extreme_abs_151_h1: AQI at t + 1 month >= 151
extreme_abs_151_h3: AQI at t + 3 months >= 151
```

Additional targets are also generated for AQI >= 201 and city-relative percentile
thresholds.

## Dataset

The main input file is:

```text
data/global_air_quality_2014_2025.csv
```

Country-specific duplicate extracts and generated processed matrices are not required to
run the configured pipeline from scratch.

The dataset is a synthetic monthly global air-quality benchmark covering:

```text
Rows: 331,920
Countries: 24
Cities: 2,305
Monthly dates: 144
Date range: 2014-01-01 to 2025-12-01
Missing values in the consolidated raw file: 0
```

The dataset contains pollutant, meteorological, and socio-environmental variables,
including:

```text
PM2.5, PM10, NO, NO2, NOx, NH3, CO, SO2, O3,
Benzene, Toluene, Xylene, AQI, AQI_Bucket,
Wind_Speed, Humidity, Deforestation_Rate,
Industry_Growth, CO2_Emission, Population_Density
```

Important note: the data should be treated as a controlled synthetic benchmark, not as
direct observational measurements from real monitoring networks.

## Main Methodology

The pipeline evaluates binary classification models for future extreme AQI events.

Each sample is a city-month observation. The models use current covariates, temporal
features, lagged values, rolling windows, interaction features, and optionally local
normalized features based on each city's past history.

The main evaluation protocols are:

```text
Random split:
  70% train, 15% validation, 15% test, stratified by target.

Temporal split:
  train:      2014-01 to 2021-12
  validation: 2022-01 to 2023-12
  test:       2024-01 to 2025-12

Leave-one-country-out:
  each country is held out as the test domain once.
  validation is sampled from the remaining training countries.
```

The leave-one-country-out protocol is the main geographic generalization benchmark.

## Mitigation Strategies

The repository includes several strategies for improving geographic transfer:

```text
Local normalized features:
  Past city-level anomaly and z-score features.

Similarity weighting:
  Training countries are weighted by environmental similarity to the held-out country.

Inverse country-size weighting:
  Each training country receives equal total sample mass.

Country-class balanced weighting:
  Each country receives balanced mass, and classes are balanced within each country when both classes are present.
```

Similarity weighting is implemented in `src/similarity_weighting.py`.

Domain weighting is implemented in `src/domain_weighting.py`.

## Metrics

Evaluation metrics are implemented in `src/evaluation.py`.

Reported metrics include:

```text
accuracy
balanced_accuracy
precision
recall
f1
macro_f1
roc_auc
pr_auc
false_alarm_rate
missed_event_rate
confusion-matrix counts: tn, fp, fn, tp
```

PR-AUC and ROC-AUC are computed from model scores when available.

Threshold-dependent metrics are computed from model predictions. For probabilistic binary
classifiers, this corresponds to the default classifier threshold, typically 0.50.

The operational analysis evaluates thresholds from 0.00 to 1.00 in steps of 0.01.

## Important Methodological Notes

The main absolute targets, such as `extreme_abs_151_h1` and `extreme_abs_151_h3`, are
constructed from future AQI values within each city.

City-relative targets, such as `extreme_city_p90_h1` and `extreme_city_p95_h1`, are
available in the pipeline but require special care because city-level percentile
thresholds should be computed only on the appropriate training reference data when used
for strict leakage-free evaluation.

Local normalized features use each city's past observations and do not use future values
for the current month.

Similarity weighting uses unlabeled covariate profiles from the held-out country. This
should be treated as a transductive setting, not as a strictly inductive deployment
scenario.
